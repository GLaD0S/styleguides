Sass

All of TTF14's CSS is written in Sass initially, and partials are included in an app.scss file. TTF14 uses four Compass imports: opacity, CSS3, transform, and border-radius. It uses responsive, fonts, grid, and tables from tt-layouts. The settings partial sets up color and layout variables. The theme import includes logo styles, general styles such as paragraphs and links, typography, wrappers and containers, header and footer styles. The pages import includes styles specific to individual pages. The nav import includes menu styles. The mobile menu import includes mobile-specific styles and the off-canvas track menu. There's a mixins import for mixins used throughout. There's also an HTML5 boilerplate import.

JavaScript

JavaScript scripts that need to be loaded include Modernizr, jQuery, Flexslider, Fastclick, and Offcanvas. Offcanvas is used for the tracks menu.

Branding

The TTF14 site has web- and mobile-specific logos in both the header and the footer. In addition, the background image included in the header and image fixed above the footer are matched to other materials for TTF14 to ensure consistency across. General Texas Tribune colors such as $trib_yellow are used in the TTF14 site for consistency. The unique track colors are also consistent across all appearances of tracks in TTF14 materials to give the eye a visual cue to the information belonging to each track.

Typography

Fonts

Fonts used in the TTF14 site include proxima-nova, rooney-sans, and rooney-web. These fonts are included with Typekit. There's a font_styles mixin that can be used to set font color, weight, size, and line height, respectively.

Headers

This is the default h1 style.

H1 headers are used mainly for page titles. The default h1 style is color $dark_navy, font weight of 500, font size of 2.25em, line height of 1.2em, 'rooney-sans' font family, a bottom margin of 0.5em, and text aligned left.

This is the default h2 style.

H2 headers are used for session names on the Schedule page and track detail page, speaker names on the track detail and Speakers pages, Sign Up: label in festival subscription, time slot on Schedule page, and Sponsors page headers. The default h2 style is color $light_navy, font weight of 800, font size of 1.65em, line height of 1.2em, letter spacing of 1px, 'proxima-nova' font family, and text transform to uppercase.

This is the default h3 style.

H3 headers are used for 'Back To Top' navigation on mobile, hashtags, session times on Schedule page, and venues on track detail pages. The default h3 style is color $dark_navy, font weight of 500, font size of 1.45em, line height of 1.2em, and 'rooney-sans' font family.

Responsive

The TTF14 site mainly uses media queries to ensure that it looks good when viewed at any size. The main media queries used are those for tablet (max-width: 799px) and for mobile (max-width: 520px). Queries are particularly used to ensure that the min height for the header is at least high enough to include the site navs. The mobilemenu import also includes the helper classes .hide-for-mobile, .hide-for-medium-down, .show-for-medium-down, and .mobile-only to display or hide for medium (tablet) and mobile sizes.

Navigation

Navigation is mainly through the site nav and the pages nav, both included in the header at the top of each page with links to different pages and to social media sites like Twitter, FB and Instagram. Register, Schedule, and Tracks images are included in the right rail and link to pages around the site to ensure ease of navigation. The logo in the footer links to the homepage.

Mobile and Tablet Navigation

At the bottom of the mobile site, Back To Top links to the top of the page. Back To Top has a class of .back-to-top.

Tablet includes the site-nav, but mobile does not. Mobile does include the pages nav menu, with each li adjusted to 50% width. Register, Schedule, and Tracks images, respectively, are also included on tablet and mobile sizes at the top of the page below the header.

Each speaker has a headshot image displayed for them on the Speakers page. If a speaker doesn't yet have an image, a generic speaker avatar image should be used. Images for speakers are entered in Sched.

The TTF14 site includes a background image in the header, and it also includes a matching image above the footer. The image above the footer is included as an image.

Paragraphs

Paragraphs with just the p tag are styled with 'rooney-web' font family, color $navy, font weight of 300, font size of 1em, and line-height of 1.35em.

Homepage

Paragraphs on the homepage that are included as part of the 'Updates' section are styled with 'proxima-nova' font family, font weight of 400, and font size of 1em.

                        
                            <article class="update">
                                <p>
                                    ...
                                </p>
                            </article>

Schedule Page

Paragraphs are used on the schedule page for session descriptions. These paragraphs are styled with 'proxima-nova' font family, font weight of 400, and line height of 1.2em.

                        
                            <div class="program-description">
                                <p>
                                    ...
                                </p>
                            </div>

Paragraphs are also used on the schedule page for track headers. These styles are designated with class .track, which floats the track name left.

                        
                            <div class="program-description">
                                <p class="track">
                                    Track Name
                                </p>
                            </div>

Venue names are styled with paragraphs with class .venue. This class floats the venue name to the left, and gives it font family of 'proxima-nova', color $dark_navy, font weight of 400, and font size of 1em.

                        
                            <div class="program-description">
                                <p class="venue">
                                    Track Venue
                                </p>
                            </div>

Speakers Page

The speakers page uses paragraphs with a class of .panel-link to style the Sessions and session names for each speaker, which gives the styles of 'rooney-sans' font family, font weight of 500, color $dark_navy, font size of 1em, line height of 1.1em and float left.

                        
                            <div class="speaker">
                                <p class="panel-link">
                                    Sessions: Session Name, Session Name...
                                </p>
                            </div>

Lists

List elements are used in the site nav for links, on the sponsors page for sponsor Twitter handles, on the homepage for list elements in Updates, in the festival wire for Updates, and on the Schedule page for session speakers.

Sponsors page

The Sponsors page uses list elements within a ul with class .handle_list to style sponsor Twitter handles.

                        
                            <ul class=.handle_list>
                                <li>
                                    Twitter handle
                                </li>
                            </ul>

Links

Links are generally styled with color #008990. They have a text decoration of none, a font size of 1em, a line height of 1em, and a hover state of text decoration underline and also color #008990.

This is a link

Schedule Page

On the Schedule page, the track names are links. The links are inside paragraphs with class .track. These links have a style of 'proxima-nova' font family, color white, font weight of 800, font size of 1.65em, letter spacing of 1px, and line height of 1.2em. These links have text transform of uppercase. On hover, the color remains white.

                        
                            <p class=track>
                                <a>
                                    Track Name Link
                                </a>
                            </p>

Day buttons on the Schedule page are links. These are styled with an anchor link inside an li of class .day-btn. These links have a hover state of background $festive_blue, color $dark_navy, text-decoration none.

                        
                            <li class=day-btn>
                                <a>
                                    Day
                                </a>
                            </li>

Site Nav

Links are used in the site nav for navigation to different pages. Style for these links is color $navy, display block, 'proxima-nova' font family, font weight of 600, and text transform to uppercase. The active state is a background of $festive_blue. The hover state is a background color of #9ed8d4, color $navy, and text decoration of none.

                        
                            <nav class="nav site-nav">
                                <ul>
                                    <li>
                                        <a>
                                        </a>
                                    </li>
                                </ul>
                            </nav>

Social Media

Twitter, Facebook, and Instagram social media icons are included in the header and footer of the page. These are Font Awesome icons. The hover state for the icons is $festive_blue. The icons are placed inside links with class of .social and given a class with their appropriate Font Awesome name.

                        
                            <a class="social">
                                <i class="icon-iconname">
                                </i>
                            </a>

Twitter Hashtags

The Twitter hashtags for the festival and for each track are included with links that open new tabs to the appropriate Twitter hashtag search page using target="_blank". The festival hashtag of #tribunefest is a link included as an h3 in the site header. The hover state is an underline of the same color. The #tribunefest link is not displayed on mobile.

                        
                            <div class=nav>
                                <h3>
                                    #tribunefest
                                </h3>
                            </div>

Each track's hashtag is included on track detail pages. On the track detail page, an h3 with a class of .hashtag is used to style the hashtag. The hover state is an underline of the same color.

                        
                            <body class=track-detail>
                                <header class="track_header">
                                    <h3 class="hashtag">
                                        #TTF14-track-name
                                    </h3>
                                </header>
                            </body>

Sched

The TTF14 site's Django backend is fed info through Sched's API. In Sched, information about sessions, speakers, speakers' images, and more is entered.
