Sass

All of TTF14's CSS is written in Sass initially, and partials are included in an app.scss file. TTF14 uses four Compass imports: opacity, CSS3, transform, and border-radius. It uses responsive, fonts, grid, and tables from tt-layouts. The settings partial sets up color and layout variables. The theme import includes logo styles, general styles such as paragraphs and links, typography, wrappers and containers, header and footer styles. The pages import includes styles specific to individual pages. The nav import includes menu styles. The mobile menu import includes mobile-specific styles and the off-canvas track menu. There's a mixins import for mixins used throughout. There's also an HTML5 boilerplate import.

JavaScript

Off-canvas JavaScript is used for the track menu.

Typography

Fonts

Fonts used in the TTF14 site include proxima-nova, rooney-sans, and rooney-web. These fonts are included with Typekit. There's a font_styles mixin that can be used to set font color, weight, size, and line height, respectively.

Responsive

The TTF14 site mainly uses breakpoints to ensure that it looks good when viewed at any size. The main breakpoints used are those for tablet (max-width: 799px) and for mobile (max-width: 520px). Breakpoints are particularly used to ensure that the min height for the header is at least high enough to include the site navs. The mobilemenu import also includes the helper classes .hide-for-mobile, .hide-for-medium-down, .show-for-medium-down, and .mobile-only to display or hide for medium (tablet) and mobile sizes.

Navigation

Navigation is mainly through the site nav and the pages nav, both included in the header at the top of each page with links to different pages and to social media sites like Twitter, FB and Instagram. Images included in the right rail also link to pages around the site, and these buttons are included at the top of the page below the header on tablet and mobile sizes to ensure ease of navigation. The festival logo included in the footer links to the homepage.

Images

Speakers

Each speaker has a headshot image displayed for them on the Speakers page. If a speaker doesn't yet have an image, a generic speaker avatar image should be used. Images for speakers are entered in Sched.

Background

The TTF14 site includes a background image in the header, and it also includes a matching image above the footer. The image above the footer is included as an image.

Logos

The TTF14 site uses different images for the desktop and mobile logos included in the header and footer.

Buttons

There are three buttons included in the right rail, and these link to the Register page, Speakers page, and display the Tracks menu.

Paragraphs

Paragraphs with just the p tag are styled with 'rooney-web' font family, color $navy, font weight of 300, font size of 1em, and line-height of 1.35em.

Homepage

Paragraphs on the homepage that are included as part of the 'Updates' section are styled with 'proxima-nova' font family, font weight of 400, and font size of 1em.
