Welcome

As living documents of Texas Tribune sites and platforms, style guides are built to visually match these projects. Each style guide uses the stylesheet from the project it records so that as a project's look evolves, the updated styles can be added in and the style guide will continue to match the project.

The goal is that these style guides provide visual references and coding examples that help with the consistent implementation of styles across a project. The style guides are meant to be welcoming and helpful to all who visit, and we strive to help each user get out of the style guides what she particularly needs.
